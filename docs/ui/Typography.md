### Typography

Semantic text components.

#### Import
```tsx
import { HeroTitle, Body, BodyLarge, H1, H2, H3Serif, H4, Eyebrow } from "../../components/ui/Typography.tsx";
```

#### Usage
```tsx
<Eyebrow>Badge text</Eyebrow>
<HeroTitle>Hero heading</HeroTitle>
<Body>Paragraph</Body>
<BodyLarge>Prominent paragraph</BodyLarge>
<H1>Section</H1>
<H2>Subsection</H2>
<H3Serif>Serif heading</H3Serif>
<H4>Caption</H4>
```

All components accept:
- class: string
- HTML attributes for their tag (e.g., id, style)
