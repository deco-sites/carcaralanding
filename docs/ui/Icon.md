### Icon

SVG sprite icon renderer.

#### Import
```tsx
import Icon, { type AvailableIcons } from "../../components/ui/Icon.tsx";
```

#### Props
- id: AvailableIcons (see `/static/sprites.svg`)
- size: number
- strokeWidth: number
- Other SVG attributes

#### Usage
```tsx
<Icon id="ChevronRight" size={20} class="text-ca-50" />
```
